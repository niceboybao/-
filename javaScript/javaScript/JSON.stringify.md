# js 与json的格式转

## JSON.stringify() 和 JSON.parse()

JS对象转换成为JSON

```javascript
JSON.stringify()
```

JSON转换成为JS

```javascript
JSON.parse()
```

## angular.fromJson() 和 angular.toJson()

AngularJs的`angular.fromJson()`方法可以把一个Json字符串中解析成一个对象，或对象数组

AngularJs的`angular.toJson()`方法，从对象到Json
